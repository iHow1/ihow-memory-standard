# v0.1 Release Scope

iHow Memory v0.1 is a public open spec draft seeking RFC-style review.

## Included

- public whitepaper
- protocol semantics
- reliability scenarios
- conformance specification direction
- namespace and security boundary notes
- license and trademark notes

## Excluded

- tool integration code
- SDKs
- runtime services
- hosted service code
- deployment recipes
- private operations
- customer-specific materials
- generated benchmark data

## Reason

The first public release should make the problem, vocabulary, boundaries, and acceptance criteria clear before publishing implementation details.
